Introduction

C programming is a general-purpose, procedural, imperative computer programming language developed in 1972 by Dennis M. Ritchie at the Bell Telephone Laboratories to develop the UNIX operating system.

C is the most widely used computer language. It keeps fluctuating at number one scale of popularity along with Java programming language, which is also equally popular and most widely used among modern software programmers.

  • C was invented to write an operating system called UNIX.
  • Today's most popular Linux OS and RDBMS MySQL have been written in C.
  • Today C is the most widely used and popular System Programming Language.
  • Most of the state-of-the-art software have been implemented using C.

Variables

Variables are basically storage area of our program.Variables have specific type,size and layout of variable's memory.
A variable's name can be composed of letters,digits and underscore'_'.It can start with a letter or an underscore.
Uppercase and Lowercase letters are distinct because C is case-sensitive.

Declaring Variables

Char:

Stores a single character and it's storage memory is 1 bit.

char myCharacter ='a'; //a->character of 1 bit Int:

Stores integer type value and it's storage memory is 2 or 4 bits.

int myInteger = 10; Float:

Stores decimal values and it's storage memory is 4 bits.

float myFloat = 1.2513; Double:

Stores floating values of 64 bits.

double myDouble = 5461.49614; String:

String is an array(collection) of characters.

char myString = "This is String";

Variable Scope

Scope of a variable is the region in the program where the defined variable exists and beyond the region the variable cannot be accessed.

Local and Global Variables

Local Variables:

Variables that are declared/defined inside a function or block are called Local Variables.These can be used within that function.Outside the function these variables are not accessible.

#include<stdio.h>
int main()
{
 int a,b,c;
 a=10;
 b=20;
 c=a+b;
 printf("Sum=%d",c);
 return 0;
}

Global Variables:

Variables that are declared outside the function are called Global Variables.These variables hold their value throughout the lifetime of the program and can be accessed by any function inside the program.
If same variable is being declared both inside and outside a function with different values,then the variable declared inside will be taken preference.

#include<stdio.h>
int g=10;
int main()
{
 int a,b;
 a=5;
 b=a+g;
 printf("Sum=%d",b);
 return 0;
}

If...Else Statement

If :

An if statement consists of a Boolean expression followed by one or more statements

#include<stdio.h>
int main()
{
 int a=10;
 if(a<20) {
  printf("a is less than 20");
  }
 return 0;
}

If Else:

An if statement can be followed by an optional else statement, which executes when the Boolean expression is false.

#include<stdio.h>
int main() {
 int a=10;
 if(a<20) {
  printf("a is less than 20");
 }
 else {
  printf("a is not less than 20");
 }
 return 0;
}

Arrays

Arrays a kind of data structure that can store a fixed-size sequential collection of elements of the same type. An array is used to store a collection of data, but it is often more useful to think of an array as a collection of variables of the same type

A specific element in an array is accessed by an index. All arrays consist of contiguous memory locations. The lowest address corresponds to the first element and the highest address to the last element.

Declaring Arrays:

type arrayName [ arraySize ];

Example:

int a[10]; //one-dimensional array

Initialising Arrays:

type arrayName [ arraySize ] = {array elements};

Example:

int a[5]={1,2,3,4,5};

For Loop

Repeats a statement or group of statements while a given condition is true. It tests the condition before executing the loop body.

#include<stdio.h>
int main() {
 int a=10;
 while(a>20) {
  printf("value of a: %d\n", a);
 }
 return 0;
}

While Loop

Repeats a statement or group of statements while a given condition is true. It tests the condition before executing the loop body.

#include<stdio.h>
int main() {
 int a=10;
 while(a>20) {
  printf("value of a: %d\n", a);
  a++;
 }
 return 0;
}
